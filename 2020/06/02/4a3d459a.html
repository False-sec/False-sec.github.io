Dockerfile

常用的几个命令

命令 作用
FROM FROM是必须的，指定一个基础镜像
WORKDIR 指定默认的工作目录,如果目录不存在会自动创建
COPY 将宿主机文件或文件夹copy进docker中,无法加载网络资源
ADD 与COPY类似，本地的tar文件会自动在容器内解压,ADD可加载网络资源
RUN 在docker镜像构建的时候执行的shell命令
CMD 在docker镜像每次启动运行时执行的命令
ENTRYPOINT 与CMD类似，使用时是数组格式[“命令”,”参数”,”文件”],当Dockerfile种存在多个ENTRYPOINT与CMD时，只有最后一个会生效
ENV 添加系统的环境变量
EXPOSE 指定暴露的端口，web镜像时常用80

在使用Dockerfile构建镜像时，FROM命令是必须的，为镜像指定一个基础镜像
